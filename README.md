# Projetos Angular

Este workspace contém quatro projetos Angular, cada um com foco em diferentes aprendizados e funcionalidades.

## Projetos

### 1. AnyBank
- Um banco digital simples.
- Demonstrou conceitos de contas, extratos e transações.
- Projeto focado em lógica financeira básica e interface de usuário.

### 2. Organo
- Um organizador de livros.
- Permite gerenciar coleções e componentes relacionados a livros.
- Projeto voltado para estrutura de componentes e rotas no Angular.

### 3. Freelando Reactive Form
- Plataforma mais complexa.
- Aprendizado profundo em `Reactive Forms` do Angular.
- Inclui formulários avançados, validações e organização de páginas.

### 4. Polpep
- Um projeto de banco com arquitetura de componentes Angular completa.
- Aprendi muito sobre composição de componentes e estrutura de aplicação.
- Abrange área financeira, contas, saldo e transações em uma interface modular.
